# 英才ブログ生成ツール

英才個別学院のブログ記事＋サムネイル画像を簡単に作成するためのツールです。

---

## 🚀 インストール方法（社員向け）

### ステップ1：Tampermonkey をインストール

下のリンクをクリックして、Chrome に Tampermonkey 拡張機能を追加してください。

👉 [Tampermonkey をインストール](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)

- 「Chrome に追加」→「拡張機能を追加」をクリック
- 右上に黒いアイコンが出ればOK

---

### ステップ2：ブログ生成ツールをインストール

下のリンクをクリックすると、Tampermonkey のインストール画面が開きます。

👉 [ブログ生成ツール v0.56.0 をインストール](https://raw.githubusercontent.com/honbueisai/blog-tools/main/blog-generator_v0.56.0.user.js)

- 「インストール」ボタンを押すだけで完了！

---

### ステップ3：装飾ツールをブックマーク

ブログの見出しやリストを装飾するツールです。  
下のリンクをブックマークしておいてください。

👉 [装飾ツール（editor-icons.html）](https://tools.eisai.org/blogs/editor-icons.html)

---

## 📝 使い方

1. [Gemini](https://gemini.google.com/app) を Chrome で開く
2. 画面左に青い丸ボタン（📝）が出るのでクリック
3. テーマ・メモ・校舎情報を入力して「Geminiへ送信して記事生成」
4. 生成されたHTMLをコピー → 装飾ツールに貼り付け → テーマを適用
5. 完成したHTMLをブログ管理画面に貼り付けて公開

---

## 🔄 アップデート方法

新しいバージョンが出たら、再度「ブログ生成ツールをインストール」のリンクをクリックするだけで上書きされます。

---

## ❓ 困ったとき

- ボタンが出ない → ページを再読み込み（⌘R）してみてください
- それでも動かない → 管理者に連絡してください
