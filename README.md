# 英才ブログ生成ツール

英才個別学院のブログ記事＋サムネイル画像を簡単に作成するためのツールです。

---

## 🚀 インストール方法（社員向け）

**所要時間：約3分**

---

### ステップ1：Google Chrome を使う

このツールは **Google Chrome** 専用です。  
Safari や Edge では動きませんので、必ず Chrome を使ってください。

Chrome がインストールされていない場合：  
👉 <a href="https://www.google.com/chrome/" target="_blank">Google Chrome をダウンロード</a>

---

### ステップ2：Tampermonkey をインストール

Tampermonkey は、Chrome に機能を追加するための拡張機能です。  
1回だけインストールすれば、以降は自動で動きます。

#### 手順

1. 下のリンクをクリック  
   👉 <a href="https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo" target="_blank">Tampermonkey をインストール</a>

2. Chrome ウェブストアが開きます

3. 画面右上の青いボタン **「Chrome に追加」** をクリック

4. 確認ダイアログが出たら **「拡張機能を追加」** をクリック

5. Chrome の右上（アドレスバーの横）に **黒い四角のアイコン** が表示されれば成功です

#### 確認方法

- Chrome 右上に黒いアイコン（四角の中に2つの丸）が出ていればOK
- 見つからない場合は、Chrome 右上の **パズルのアイコン** をクリック → Tampermonkey を探してピン留め

---

### ステップ3：ブログ生成ツールをインストール

#### 手順

1. 下のリンクをクリック  
   👉 <a href="https://raw.githubusercontent.com/honbueisai/blog-tools/main/blog-generator_v0.56.0.user.js" target="_blank">ブログ生成ツール v0.56.0 をインストール</a>

2. 新しいタブが開き、**Tampermonkey のインストール画面** が表示されます

3. 画面に「このスクリプトをインストールしますか？」のような内容が表示されます

4. 右側または下部にある **「インストール」** ボタンをクリック

5. 「スクリプトがインストールされました」と表示されれば完了

#### うまくいかない場合

- Tampermonkey がインストールされていない可能性があります → ステップ2に戻る
- ページが真っ白になる場合 → ページを再読み込み（⌘R または Ctrl+R）

---

### ステップ4：装飾ツールをブックマーク

ブログの見出しやリストをおしゃれに装飾するためのツールです。

#### 手順

1. 下のリンクをクリック  
   👉 <a href="https://tools.eisai.org/blogs/editor-icons.html" target="_blank">装飾ツール（editor-icons.html）</a>

2. ツール画面が開いたら、ブラウザのブックマークに追加  
   - Mac：⌘ + D  
   - Windows：Ctrl + D

3. 毎回このブックマークから開けばOK

---

- [ ] ブログ生成ツールをインストールした

## 📝 ブログ記事の作り方

### STEP 1：Gemini を開く

1. Chrome で下のリンクを開く  
   👉 <a href="https://gemini.google.com/app" target="_blank">Gemini を開く</a>

2. 画面が表示されたら **3〜5秒待つ**

3. 画面の **左端** に **青い丸ボタン（📝マーク）** が表示されます

4. そのボタンをクリックすると、右側にツールパネルが開きます

#### ボタンが出ない場合

- ページを再読み込み（⌘R または Ctrl+R）してみてください
- それでも出ない場合は、Tampermonkey アイコンをクリックして、スクリプトが有効になっているか確認

---

### STEP 2：教室情報を設定（初回のみ）

ツールパネルの上部に **「⚙️ 教室情報設定」** という折りたたみがあります。

1. クリックして開く

2. 以下を入力：
   - **校舎名**：例）武蔵新城校
   - **室長名**：例）湯浅
   - **CTAリンク先URL**：お問い合わせページのURL
   - **電話番号**：例）044-xxx-xxxx

3. **「教室情報を保存」** ボタンを押す

4. 「教室情報を保存しました」と表示されればOK

※ この設定は1回だけ。次回以降は自動で読み込まれます。

---

### STEP 3：ブログを生成する

1. **「ブログのテーマ」** を入力  
   例）「冬休みの勉強の進め方」「定期テスト対策のコツ」

2. **「メモ」** に補足情報を入力（任意）  
   例）「中1〜中3向け」「ワークの使い方を入れたい」

3. **「Geminiへ送信して記事生成」** ボタンをクリック

4. 数十秒待つと、Gemini が記事を生成します

5. 生成が完了すると、ツールパネルに  
   **「✅ ブログ記事の生成が完了しました」** と表示されます

6. 赤いボタン **「▶ ブログHTMLをコピーする」** をクリック

7. 「✅ ブログHTMLをコピーしました」と表示されればOK

---

### STEP 4：装飾ツールでデザインを適用

1. ブックマークしておいた **装飾ツール** を開く  
   👉 <a href="https://tools.eisai.org/blogs/editor-icons.html" target="_blank">装飾ツール</a>

2. 画面左側の大きな入力エリアをクリック

3. **⌘ + V（Mac）** または **Ctrl + V（Windows）** で貼り付け

4. 画面上部の **「テーマ一覧」** から好みのデザインを選ぶ

5. **「テーマを適用」** ボタンをクリック

6. 右側に装飾されたHTMLが表示されます

7. 右側のHTMLを **全選択（⌘A）→ コピー（⌘C）**

---

### STEP 5：ブログ管理画面に貼り付け

1. ブログ管理画面（WordPress など）を開く

2. 記事の本文欄に **貼り付け（⌘V）**

3. プレビューで確認して、問題なければ **公開**

---

## 🖼 サムネイル画像の作り方（オプション）

ブログ記事を生成した後、同じツールでサムネイル画像も作れます。

1. ブログHTMLをコピーした後、画面を下にスクロール

2. **「🖼 サムネイル画像生成」** セクションが表示されます

3. **画像スタイル**（実写・アニメなど）を選ぶ

4. **訴求スタイル**（共感・驚きなど）を選ぶ

5. **「▶ 画像生成用プロンプトを作成」** をクリック

6. Gemini がプロンプトを生成します

7. 完了したら **「このプロンプトで画像を生成する」** をクリック

8. Gemini が画像を生成します

---

## 🔄 アップデート方法

新しいバージョンが出たら、管理者から連絡があります。

1. 再度「ブログ生成ツールをインストール」のリンクをクリック
2. 「インストール」を押すだけで上書きされます

## ❓ よくある質問

### Q. 青い丸ボタンが出ない

- ページを再読み込み（⌘R）してみてください
- Tampermonkey アイコンをクリックして、スクリプトが有効か確認
- それでも出ない場合は管理者に連絡

### Q. 「教室情報を保存」しても消える

- ブラウザのプライベートモード（シークレットモード）では保存されません
- 通常モードで Chrome を使ってください

### Q. 装飾ツールでデザインが崩れる

- 最初に生成したHTMLからやり直してみてください
- CTA（お問い合わせボタン部分）は自動で保護されているので、触らないでください

### Q. 画像が生成されない

- Gemini の画像生成モード（バナナマーク）がオンになっているか確認
- 思考モードもオンにしてください
